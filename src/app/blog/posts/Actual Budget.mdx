## Quick start

Clone the git repository:

<CodeBlock compact marginBottom="16" codes={[
  {
    code: "docker stop actualbudget
docker rm actualbudget
",
    language: "bash"
  }
]} />

Install the necessary dependencies:

<CodeBlock compact marginBottom="16" codes={[
  {
    code: "docker pull actualbudget/actual-server:latest",
    language: "bash"
  }
]} />

Start the local development server:

<CodeBlock compact marginBottom="16" codes={[
  {
    code: "docker run -d \
  --name actualbudget \
  -p 5006:5006 \
  -v /home/pi/actual-data:/data \
  actualbudget/actual-server:latest
",
    language: "bash"
  }
]} />
